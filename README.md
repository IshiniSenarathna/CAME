# CAME

## Project Overview


## Features

- Interactive 3D visualizations
- Step-by-step manufacturing process tutorials
- Real-time simulation capabilities
- Educational resource management

## Installation

1. Clone the repository:
```bash
git clone https://github.com/IshiniSenarathna/CAME.git
```

2. Navigate to project directory:
```bash
cd CAME
```

## Usage

Run the Jupyter notebook to start the application:
```bash
jupyter notebook CAME.ipynb
```

## Technologies Used

- Python
- Jupyter Notebook
- CAD/CAM Libraries
- 3D Visualization Tools
